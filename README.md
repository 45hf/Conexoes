# Conexoes 

Bot automatizado que lê links de uma planilha, encurta com ShrinkEarn e publica automaticamente em canais do Telegram. Feito com Python.

## Tecnologias
- Telegram Bot API
- ShrinkEarn API
- Google Sheets API
- Replit + Render

## Setup
- Crie o arquivo `config.json` com suas chaves.
- Instale as dependências.
- Execute `main.py`.
